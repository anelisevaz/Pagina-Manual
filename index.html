Meu relato atual:

Estude e muito! Coloque em prática o conhecimento adquirido, por mais simples que seja.

Por exemplo: estou estudando operadores aritméticos. Faça o exercício (uma calculadora simples com operadores aritméticos) com a linguagem de programação escolhida e "suba" esse projeto no GitHub.

Assim, além de colocar em prática o conhecimento, também aprenderá a utilizar o editor de texto e fazer git para disponibilizar o projeto no GitHub.
